# Icon Prompts for New Portfolio Cards

Use these in ChatGPT image generation so icons 11-14 match your current set.

## Shared style lock (paste this before each prompt)
- 1024x1024 square PNG
- Cute chibi mascot illustration, high polish, game-like
- Thick dark navy outlines, smooth glossy shading, soft highlights
- Centered composition (character + tools), subject fills about 70% of frame
- Dark radial vignette background with a subtle colored glow behind subject
- Small floating sparkles/mini-icons around the character
- Colorful but controlled palette; match accent color noted per project
- No text, no letters, no numbers, no logo marks, no watermark

## 11) Docket Whale (accent: #2ED3A5)
Create a cute futuristic robot restaurant-ops captain in a small manager cap, confidently holding a tablet with order tiles and pointing at a floating kitchen workflow board. Add mini symbols around it: serving bell, ticket slip, stock crate, and routing arrows between station nodes. Use teal/green accent lighting, dark navy outlines, glossy cartoon shading, soft glow, and the same mascot style as polished portfolio game icons. Centered, clean, no text.

## 12) Roley (accent: #4FC3F7)
Create a friendly scout robot in a casting spotlight, holding a clipboard and a magnifier while scanning floating talent cards and alert icons. Add small orbiting elements: notification bell, calendar badge, target marker, and a paper-plane alert symbol. Use cool blue accents, dark vignette background, thick navy outlines, glossy chibi style, and sparkly motion cues to suggest automated scanning. Centered, no text.

## 13) Psyche (accent: #8DA2FF)
Create a thoughtful AI analyst mascot (cute robot) examining a glowing layered mind-map hologram above its hands. Surround it with floating cognitive symbols: puzzle piece, balance scale, neural nodes, and a citation card icon. Use indigo/blue accents with subtle cyan highlights, dark radial glow background, thick outlines, and polished cartoon rendering consistent with existing portfolio icons. Centered composition, no text.

## 14) Grazi Assistant (accent: #FFB15C)
Create a cheerful personal-assistant robot juggling daily ops items: inbox envelope, checklist sheet, calendar page, coin/expense chart, and a reminder bubble. Add small orbiting automation cues like clock, pipeline arrows, and a notification badge. Use warm orange/gold accents, dark vignette background, thick navy outlines, glossy chibi shading, and matching high-polish mascot style. Centered, no text.
