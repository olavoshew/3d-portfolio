# 3D Portfolio

A spatial portfolio site built with Three.js where 10 project cards float in 3D space. Click a card and the camera moves to it, revealing project details in an animated overlay. Works on desktop and mobile with touch input and optional ambient audio.

## Tech Stack

- Three.js (3D rendering, camera rig, raycaster interaction)
- Vite (dev server, production bundler, tree-shaking)
- GSAP (HTML overlay animations, lazy-loaded)
- Vanilla JavaScript (no framework)
- Web Audio API (ambient audio layer)
- Deployed on Vercel
